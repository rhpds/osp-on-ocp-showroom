= From Proprietary to Powerful: Accelerate Your VMware Migration to the Red Hat Hybrid Cloud

Welcome to the VMware to Red Hat Hybrid Cloud migration workshop!

== What you'll learn

In this workshop, you will:

* Explore the pre-deployed migration infrastructure components (Conversion Host, networks, security groups)
* Review existing Ansible Automation Platform job workflows configured for VMware to OpenStack migration
* Configure Ansible Automation Platform job workflows to perform VM seeding and final cutover
* Experience a two-phase migration approach that minimizes downtime during maintenance windows

== Who this is for

This workshop is designed for **Infrastructure Engineers and VMware Administrators** who want to learn how to migrate workloads from VMware to Red Hat OpenStack Services on OpenShift using Ansible Automation Platform.

**Experience level**: Intermediate

== Prerequisites

Before starting this workshop, you should have:

* Basic understanding of VMware vCenter and virtual machine management
* Familiarity with OpenStack concepts (compute, networking, storage)
* Basic knowledge of Ansible Automation Platform
* Experience with Linux command line operations

== Workshop environment

Your lab environment is fully pre-configured with all the necessary infrastructure for VMware to OpenStack migration:

* **Red Hat OpenShift cluster** hosting OpenStack services
* **Red Hat OpenStack Services on OpenShift (RHOSO)** as the target cloud platform
* **Ansible Automation Platform** for migration orchestration
* **VMware vCenter** as the source environment
* **Conversion Host** deployed in OpenStack for efficient data transfer
* **Bastion host** for command-line access
